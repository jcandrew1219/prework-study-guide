Git

  • pwd: display current directory
  • ls: displays list of files and folders in current directory
  • cd: change directory
  • cd ..: change backwards in direcctory
  • mkdir: creates new directory(folder)
  • touch: creates file inside directory
  • git status: checks current branch
  • git add -A - add all changes in branch
  • git checkout -b feature/starter-code - move working branch to new
  • branch, create new branch called feature/starter-code
  • git commit -m "add starter code": updates code changes
  • git pull origin main: pull main branch from origin to the local environment
  • git push origin feature/starter-code: push the feature branch to origin
